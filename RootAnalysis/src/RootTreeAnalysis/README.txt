RootTreeAnalysis is designed to be used with:
1 Raw Root File
or
1 Recon Root File
or
1 Monte Carlo Root File
or
1 Raw Root File and 1 Recon Root File
or
1 Raw Root File and 1 Recon File and 1 Monte Carlo File
or
any other combination...

See RootTreeAnalysis.h for more information

To Use RootTreeAnalysis:

Be sure you have the digiRootData, reconRootData, and mcRootData libraries
available on your system.
